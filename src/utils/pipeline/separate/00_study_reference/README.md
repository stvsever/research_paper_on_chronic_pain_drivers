# Stage 00 - Study Reference

Read-only reference material for the chronic-pain diary and the cognitive-affective model of the
interruptive function of pain (Eccleston and Crombez, 1999). The analysis does not use acceptance
as an analytic node. Source documents are kept locally in `src/other/legacy_dump_outlook/` and
`src/utils/study_materials/`; private documents are excluded from version control.
